This is a port of [url=https://www.nexusmods.com/skyrim/mods/79009]Unique Region Names USLEEP[/url] to Special Edition, intended for use with [url=https://www.nexusmods.com/skyrimspecialedition/mods/49698]Regional Save Names[/url].


[heading]Features[/heading]
[list]
[*] Relabel exterior locations from [i]Skyrim[/i] to their actual region name.
[*] Provide the region names required by [url=https://www.nexusmods.com/skyrimspecialedition/mods/49698]Regional Save Names[/url].
[*] Water flows are updated for Special Edition using the [url=https://www.nexusmods.com/skyrimspecialedition/mods/5221]Fix Oldrim Mods Waterflow[/url] SSEEdit script.
[*] The plugin is [url=https://www.creationkit.com/fallout4/index.php?title=Data_File#Light_Plugin]flagged as ESL[/url] and doesn't count against the ESP limit.[/list]

[heading]Installation[/heading]
[list]
[*] Install using your mod manager of choice, [url=https://www.nexusmods.com/skyrimspecialedition/mods/6194]Mod Organizer 2[/url] is recommended.
[*] Use [url=https://www.nexusmods.com/skyrimspecialedition/mods/164]SSEEdit[/url] to make sure changes to cell records don't override more important features from other mods: see [url=https://tes5edit.github.io/docs/5-conflict-detection-and-resolution.html]xEdit Conflict Detection and Resolution[/url] and [url=https://www.reddit.com/r/skyrimmods/wiki/patching]Patching[/url] to know more.[/list]
